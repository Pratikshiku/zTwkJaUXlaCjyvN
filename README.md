## Campus Lost and Found Claim System

> [Click me download source code / Try it online](https://www.devquizdone.online/detail/3abe338917fd48108aa1970263c2bfcc/ghb20250917) 

### 1. Related Technologies
- Backend: `Java`, `JavaWeb` / `Springboot`.
- Frontend: `Vue`, `HTML` / `CSS` / `Javascript`, etc.
- Database: `MySQL`

### 2. Related Software (Any one of the listed software can be used to run)
- `IDEA`
- `Eclipse`
- `Visual Studio Code(VScode)`
- `Navicat`
- etc.

### 3. Function Description
The system is divided into (roles): regular users, administrators.

**Regular User Functions:**
1. Login
2. Register
3. Homepage
4. Forum Information
5. Announcement Information
6. Lost and Found
7. Lost Item Notice
8. Personal Center
9. Lost Item Claim
10. Lost and Found
11. Lost Item Claim
12. Lost Item Notice


**Administrator Functions:**
1. Login
2. Personal Center (change password, personal information)
3. Administrator Management
4. User Management
5. Forum Management
6. Announcement Information Management
7. Lost and Found Management
8. Lost Item Claim Management
9. Lost Item Notice Management
10. Lost Item Claim Management
11. Announcement Type Management
12. Lost Item Type Management
13. Lost Item Type Management
14. Carousel Image Management
